# Study Companion Frontend

React frontend for the AI Study Companion transcript generation and validation UI.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

The app will be available at http://localhost:3001

## Features

- Generate realistic tutor-student conversation transcripts
- View and compare multiple transcripts
- Validate and rate transcript quality
- Approve transcripts for use in the system

## Configuration

The frontend is configured to proxy API requests to the backend at `http://localhost:3000`. Make sure the backend is running.
